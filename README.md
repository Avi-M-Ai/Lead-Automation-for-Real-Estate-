# מערכת אוטומציית מעקב לידים (ללא קוד)

[cite_start]מדריך מקיף להקמת מערכת מעקב לידים אוטומטית עבור סוכני נדל"ן. המערכת קולטת לידים, שולחת התראות בזמן אמת לטלגרם ומנהלת סדרת הודעות מעקב לאורך 7 ימים – ללא צורך בתכנות ובעלות של 0 ש"ח[cite: 196, 198, 201, 205].

## תכונות עיקריות
* [cite_start]**התראות זמן-אמת:** קבלת התראה מיידית בטלגרם ברגע שליד ממלא טופס[cite: 200, 330].
* [cite_start]**סדרת Follow-up אוטומטית:** ניהול רצף של 4 הודעות מעקב לאורך 7 ימים[cite: 205, 333].
* [cite_start]**ללא קוד ובחינם:** שימוש בכלים חינמיים כמו Make.com, Telegram ו-Google Forms[cite: 197, 198, 206].
* [cite_start]**זמן הקמה:** הטמעה מהירה של המערכת בכ-45 דקות בלבד[cite: 335].

## איך זה עובד
1. [cite_start]**קליטת ליד:** הליד ממלא טופס (Google Forms או Typeform)[cite: 200, 251].
2. [cite_start]**Webhook:** המידע נשלח באמצעות Webhook למערכת ה-Make.com שלך[cite: 240, 252].
3. [cite_start]**התראה בטלגרם:** הודעה נשלחת באופן מיידי לחשבון הטלגרם שלך[cite: 200, 330].
4. [cite_start]**רצף מעקב:** המערכת משתמשת במודול 'Sleep' ב-Make.com כדי לתזמן 4 הודעות מעקב לאורך 7 ימים[cite: 270, 273, 271].

## שלבי הטמעה (סיכום)
1. [cite_start]**הקמת הבוט:** יצירת בוט דרך BotFather וקבלת ה-Token וה-Chat ID[cite: 208, 232].
2. [cite_start]**הגדרת Make.com:** יצירת תרחיש (Scenario) עם Webhook וחיבור מודול Telegram[cite: 239, 240, 259].
3. [cite_start]**בניית רצף:** הוספת מודולי 'Sleep' לעיכוב הודעות המעקב[cite: 273, 285].
4. [cite_start]**חיבור טופס:** קישור הטופס ל-Webhook באמצעות Google Apps Script או הגדרות Typeform[cite: 251, 292, 303].
5. [cite_start]**בדיקה:** שליחת טסט לווידוא הגעת ההודעות[cite: 310].
